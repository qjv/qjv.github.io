Quick Start

1. Include the Library

Add the CSS and JavaScript files to your HTML page:

<!-- CSS -->
<link rel="stylesheet" href="https://qjv.dev.br/armory/styles/armory.css">

<!-- JavaScript (load in this order) -->
<script src="https://qjv.dev.br/armory/src/palette-mapping.js"></script>
<script src="https://qjv.dev.br/armory/src/data-fetcher.js"></script>
<script src="https://qjv.dev.br/armory/src/tooltip.js"></script>
<script src="https://qjv.dev.br/armory/src/renderer.js"></script>
<script src="https://qjv.dev.br/armory/src/build-parser.js"></script>
<script src="https://qjv.dev.br/armory/src/armory.js"></script>

2. Add Embeds to Your HTML

Use data attributes to specify what you want to embed:

Single Item
<div data-armory-embed="items" data-armory-ids="30704"></div>
Multiple Skills
<div data-armory-embed="skills" data-armory-ids="5492,5495,5502"></div>

3. That's It!

The library auto-initializes when the DOM is ready. All embeds will be rendered automatically.

Configuration

New Format (Recommended)

Configure the library before loading it by setting window.GW2Armory:

<script>
window.GW2Armory = {
  // Language (en, fr, de, es, zh, ru)
  lang: 'en',

  // Cache settings
  cache: {
    enabled: true,
    duration: 7 * 24 * 60 * 60 * 1000, // 7 days in milliseconds
    prefix: 'gw2armory_'
  },

  // API settings
  api: {
    base: 'https://api.guildwars2.com/v2',
    timeout: 10000
  },

  // Tooltip settings
  tooltip: {
    delay: 100,      // ms before showing tooltip
    maxWidth: 350,   // max width in pixels
    offset: 10       // offset from cursor in pixels
  },

  // Debug mode
  debug: true
};
</script>

Old Format (Backward Compatible)

The library also supports the old GW2A_EMBED_OPTIONS format:

<script>
document.GW2A_EMBED_OPTIONS = {
  lang: 'en',
  persistToLocalStorage: true,
  forceCacheClearOnNextRun: '1'  // Change this value to clear cache
};
</script>
Note: Both formats work simultaneously. The new format takes precedence if both are defined.

Configuration Options

Option Type Default Description
lang string 'en' Language code: en, fr, de, es, zh, ru
cache.enabled boolean true Enable/disable localStorage caching
cache.duration number 604800000 Cache duration in milliseconds (7 days)
cache.prefix string 'gw2armory_' Prefix for localStorage keys
api.base string GW2 API URL Base URL for GW2 API
api.timeout number 10000 API request timeout in milliseconds
tooltip.delay number 100 Delay before showing tooltip (ms)
tooltip.maxWidth number 350 Maximum tooltip width in pixels
tooltip.offset number 10 Tooltip offset from cursor (pixels)
debug boolean true Enable console logging

Embed Types

Items

Embed items with optional customization for stats, skins, upgrades, and infusions.

Basic Item
<div data-armory-embed="items" data-armory-ids="30704"></div>
Item with Custom Stat (Berserker's)
<div data-armory-embed="items"
     data-armory-ids="77482"
     data-armory-77482-stat="584"></div>
Item with Custom Skin
<div data-armory-embed="items"
     data-armory-ids="30699"
     data-armory-30699-skin="3709"></div>
Fully Customized Item (Stat + Upgrades + Infusions)
<div data-armory-embed="items"
     data-armory-ids="77482"
     data-armory-77482-stat="584"
     data-armory-77482-upgrades="24615,24868"
     data-armory-77482-infusions="49428,49428"></div>
Inline with Wiki Link
<div data-armory-embed="items"
     data-armory-ids="30704"
     data-armory-inline-text="wiki"></div>
Blank Slot
<div data-armory-embed="items"
     data-armory-ids="-1"
     data-armory-blank-text="Empty Slot"></div>

Skills

Embed skills with tooltips showing damage, cooldowns, and other facts.

Multiple Skills

     data-armory-ids="5492,5495,5502,5508"></div>
Finisher
<div data-armory-embed="skills"
     data-armory-ids="5813"
     data-armory-inline-text="wiki"></div>

Traits

Embed traits with tooltips showing their effects.

Multiple Traits

Data Attributes Reference

Common Attributes (All Types)

Attribute Values Description
data-armory-embed REQUIRED items, skills, traits, specializations, amulets, builds Type of embed
data-armory-ids REQUIRED Comma-separated IDs or chat code Item IDs to embed. Use -1 for blank slots
data-armory-size Number (pixels) Icon size. Default: 40
data-armory-inline-text "wiki", "gw2spidy", or custom text Show name next to icon. "wiki" creates wiki link
data-armory-blank-text Any text Custom text for blank slots (id: -1)

Item-Specific Attributes NEW

Attribute Example Description
data-armory-{ID}-stat data-armory-77482-stat="584" Apply custom stat (itemstat ID). 584 = Berserker's
data-armory-{ID}-skin data-armory-30699-skin="3709" Apply custom skin (skin ID). Changes icon appearance
data-armory-{ID}-upgrades data-armory-77482-upgrades="24615,24868" Pre-populate runes/sigils (item IDs)
data-armory-{ID}-infusions data-armory-77482-infusions="49428,49428" Pre-populate infusion slots (item IDs)
data-armory-{ID}-upgrade-count data-armory-1379-upgrade-count='{"24815":6}' Specify upgrade quantities (JSON format)

Specialization Attributes

Attribute Example Description
data-armory-{ID}-traits data-armory-55-traits="2071,2085,2128" Selected trait IDs (3 traits, one per tier)

Finding IDs

  • Item IDs: Use GW2TP or the GW2 Wiki
  • Skill/Trait IDs: Check the GW2 Wiki (shown in API code boxes)
  • Itemstat IDs: Check GW2 API Itemstats (584=Berserker's, 1163=Viper's, etc.)
  • Skin IDs: Check GW2 API Skins
  • Build Codes: Copy from in-game chat using the build template feature

Backward Compatibility

100% Compatible! This library is fully backward compatible with the original armory-embeds library.

What Works Out of the Box

  • ✅ All data attributes remain exactly the same
  • ✅ Old config format (document.GW2A_EMBED_OPTIONS) still works
  • ✅ Old CSS classes (.gw2a-items-embed, etc.) still work
  • ✅ data-armory-inline-text="gw2spidy" supported (shows plain text)
  • ✅ Same embed types: items, skills, traits, specializations, amulets

Migration Guide

To migrate from the old library:

  1. Replace the old script tag: 
    <!-- OLD -->
<script src="https://unpkg.com/armory-embeds@^0.x.x/armory-embeds.js"></script>

<!-- NEW -->
<link rel="stylesheet" href="https://qjv.dev.br/armory/styles/armory.css">
<script src="https://qjv.dev.br/armory/src/palette-mapping.js"></script>
<script src="https://qjv.dev.br/armory/src/data-fetcher.js"></script>
<script src="https://qjv.dev.br/armory/src/tooltip.js"></script>
<script src="https://qjv.dev.br/armory/src/renderer.js"></script>
<script src="https://qjv.dev.br/armory/src/build-parser.js"></script>
<script src="https://qjv.dev.br/armory/src/armory.js"></script>
  2. That's it! All your existing embeds will continue to work.
  3. (Optional) Migrate config to new format for better control.
  4. (Optional) Update CSS selectors to use new class names.

Missing Features from Original Library

The following features from the original armory-embeds library are not implemented:

Character Embeds

The original library supported embedding character profiles. This feature is not available in this library.

<!-- NOT SUPPORTED -->
<div data-armory-embed="character" data-armory-name="Character.1234"></div>

Why Not Included?

  • Character embeds required API keys with account access
  • Privacy concerns with displaying account data
  • Complexity of maintaining character state
  • Rarely used feature compared to item/skill embeds

New Features Not in Original

This library adds several features that the original didn't have:
  • ✨ Build Templates: Full support for in-game build codes
  • ✨ Custom Stats: Show items with specific stat combinations
  • ✨ Custom Skins: Display items with different appearances
  • ✨ Pre-filled Upgrades: Show items with runes/sigils already equipped
  • ✨ Pre-filled Infusions: Show items with infusions already slotted
  • ✨ Compact Build Display: Optimized view for build templates
  • ✨ Copy Build Code: One-click copy of build templates
  • ✨ Enhanced Tooltips: Item IDs and wiki links in all tooltips
